# インタフェース仕様(9軸センサ)
ヒアラブルデバイスの内部では、20ms周期(フレーム)でデータを収集。
BLEでスマートフォンに送る際に、5フレーム(20ms×5)にまとめて100ms毎に送信。

## 加速度センサの仕様
|項目|値|
|:--|:--|
|レンジ|±8g|
|分解能|16bit|
|周期|20ms|
|感度|4096LSB/g|

### データレンジ
|HEX|DEC|レンジ[g]|
|:--|:--|:--|
|7FFF|32767|7.999755859|
|7FFE|32766|7.999511719|
|*|*|*|
|2|2|0.000488281|
|1|1|0.000244141|
|0|0|0|
|FFFF|-1|-0.000244141|
|FFFE|-2|-0.000488281|
|*|*|*|
|8002|-32766|-7.999511719|
|8001|-32767|-7.999755859|
|8000|-32768|-8|
